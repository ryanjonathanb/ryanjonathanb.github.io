y_describe.py

DOWNLOAD

A Python-based parser for Yahoo! Finance stocks data. It processes the historical prices CSV file and outputs various useful metrics (e.g. standard deviation of changes, histogram)

usage

Make sure you have your Python intrepreter and Pandas library installed.

Open your command prompt and navigate to where you keep the script.

type:

python y_describe.py path_to_yahoo_csv_file

faq

What version of Python is this in?

Python 2.7

What file can be used as input?

The script only accepts Yahoo! Finance historical CSV files at the moment.

How to see license?

Use --license as argument
